# 🪨📄✂️ Rock-Paper-Scissors AI Trainer

Play Rock-Paper-Scissors against an AI that learns your moves!  
It predicts your next choice based on frequency and counters it intelligently.

## 💻 How to Run
```bash
python rock_paper_scissors_ai_trainer.py
```
Example Gameplay
```bash
🪨📄✂️ Rock-Paper-Scissors AI Trainer 🧠
Your move (rock/paper/scissors): rock
🤖 AI wins! (AI chose paper)
Score → You: 0 | AI: 1 | Draws: 0
```
